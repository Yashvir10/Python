# Hotel Management System

The Hotel Management System is a console-based application developed in Python using Object-Oriented Programming (OOP) principles.
The project is designed to simplify basic hotel operations such as managing customer details, calculating room rents, generating restaurant bills, laundry 
charges, and game service costs. Users can enter customer information, select different room types based on their budget, order food from the restaurant menu, 
avail laundry and gaming services, and generate a detailed bill containing all charges along with additional service fees. The system provides a menu-driven 
interface that is easy to use and helps beginners understand core Python concepts such as classes, objects, constructors, methods, loops, conditional statements,
user input handling, and billing logic. This project serves as an excellent learning resource for students who want to strengthen their understanding of Python 
programming and OOP while building a real-world application. Future enhancements can include database integration, graphical user interfaces, online booking 
functionality, payment gateway support, and web-based deployment using modern frameworks such as FastAPI or Django.
